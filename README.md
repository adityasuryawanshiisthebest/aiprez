# AIPREZ

AIPREZ is a dark futuristic React dashboard for an AI Academic Presentation System.

## Run Locally

From this folder:

```bash
python3 -m http.server 4173
```

Then open:

```text
http://localhost:4173
```

## Screens

- Dashboard
- Create Presentation
- Humanizer
- Live Notes
- Presentation Analyzer
- Presentation Viewer
- Account Settings
